# **App Name**: Cake Creation Station

## Core Features:

- Cake Selection: Select pre-designed cake options and view details.
- Custom Cake Builder (AI): A step-by-step "tool" for custom cake design, offering options to specify cake flavor, frosting, toppings, and decorations; feature can guide users in designing cakes tailored to their specifications by adapting advice to the chosen options
- Shopping Basket: Add selected cakes to a virtual shopping basket.
- Guest Checkout: Guest checkout.
- User accounts: User sign-up, login and display of loyalty points (2% return per transaction)
- Shipping Options: Shipping in 2-3 working days from checkout or 2 working days for collection
- Order Confirmation: Send confirmation email to user and order email to admin on checkout.

## Style Guidelines:

- Primary color: Soft beige (#F5F5DC) for a neutral and warm base.
- Background color: Light beige (#FDFDF5), slightly desaturated to ensure that the primary color's buttons are visible
- Accent color: Dusty rose (#D8BFD8) to give a gentle contrast to the neutral colors.
- Body and headline font: 'PT Sans' (sans-serif), offering a balance of modernity and warmth that supports readability across the platform
- Simple line icons for cake types, customization options, and basket interactions, ensuring clarity without visual clutter.
- Clean and minimalist layout to focus user attention on cake selection and customization, and ensure effortless navigation. A grid format would showcase the cakes and design components effectively.
- Subtle animations on adding items to the basket and during the cake building process to enhance user engagement without distracting from the core experience.
